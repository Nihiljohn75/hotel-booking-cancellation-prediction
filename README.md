# Hotel Booking Cancellation Prediction

## 📌 Project Overview
This project predicts whether a hotel booking will be cancelled using Machine Learning techniques. The objective is to help hotel management identify potential cancellations and improve revenue management.

## 🎯 Objective
- Predict hotel booking cancellations.
- Analyze factors influencing cancellations.
- Build and evaluate a classification model.

## 📂 Dataset
The dataset contains hotel booking information such as:
- Lead Time
- Market Segment
- Room Type
- Previous Cancellations
- Special Requests
- Booking Status

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## 📊 Project Workflow
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Building
- Model Evaluation

## 📈 Results
The model predicts hotel booking cancellations using machine learning, helping hotels make data-driven decisions to reduce revenue loss.
